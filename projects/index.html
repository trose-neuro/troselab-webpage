Current Projects and Platforms

Project 1.1 - Feedback Alignment in Visual Cortex Current Project

Chronic two-photon imaging and deep receptive-field models test how higher-order feedback stabilizes representations across dLGN, V1, and LM.

representational-drift feedback visual-cortex
Project 1.2 - Thalamocortical Stabilization in dLGN Current Project

Tests whether cortical feedback stabilizes long-term dLGN responses despite structural convergence in retinothalamic input.

thalamus plasticity dlgn
Project 2 - Experience in V1 versus CA1 Current Project

Compares representational stability in visual cortex and hippocampus after sparse manipulation of plasticity-related genes during active and passive virtual reality.

v1 ca1 experience
Project 3 - RSC Memory Stabilization Current Project

Closed-loop learning and memory paradigms with miniature two-photon imaging test how spatial and contextual signals stabilize in retrosplenial cortex.

retrosplenial-cortex learning memory
Project 4 - Natural Vision and Active Orienting Current Project

Freely moving visual orienting assays with head-mounted imaging, gaze tracking, and scene reconstruction test peri-saccadic processing.

natural-vision orienting closed-loop-behavior
Project 5 - Landmark Affordance Coding Current Project

Quantifies how RSC and V1 encode landmark utility (stability, value, salience) in gaze-contingent closed-loop navigation.

affordance landmarks navigation
Project 6 - Uncertainty and Neuromodulation Current Project

Tests how uncertainty-dependent states and cholinergic neuromodulation regulate representational drift across scales, including spatial transcriptomics.

uncertainty neuromodulation transcriptomics
Project 7 - Latent Learning Circuits Current Project

Multisensory preconditioning in virtual reality resolves circuit mechanisms of unsupervised cue association and transfer in cortical memory systems.

latent-learning preconditioning multisensory
