Projects

This page summarizes the group’s current project portfolio based on the tenure evaluation dossier. Projects are organized around representational stability, learning and memory, natural vision, and latent learning, with linked technology development.

representational-drift natural-vision learning memory infrastructure

Current Focus (Projects 1-4)

Project 1.1 - Feedback Alignment in Visual Cortex Goal I / Current

Chronic two-photon imaging and deep nonlinear receptive-field models to test how higher-order feedback stabilizes representations across dLGN, V1, and LM. Milestone target from dossier: mid-2025.

representational-drift feedback visual-cortex
Project 1.2 - Thalamocortical Stabilization in dLGN Goal I / Current

Tests whether cortical feedback stabilizes long-term thalamic responses despite structural convergence in retinothalamic input. Connected to DFG SPP 2411 support.

thalamus plasticity dlgn
Project 3 - RSC Memory Stabilization Goal II / Current

Closed-loop learning and memory paradigms with miniature two-photon imaging to study stabilization of spatial and contextual signals in retrosplenial cortex. Funded in the SFB1089 context (D09).

retrosplenial-cortex learning memory
Project 4 - Natural Vision and Active Orienting Goal III / Current

Freely moving visual orienting assays with head-mounted imaging, gaze tracking, and pose/scene reconstruction to study peri-saccadic processing.

natural-vision orienting closed-loop-behavior

Current Expansion (Projects 2, 5-7 and Platforms)

Project 2 - Experience in V1 versus CA1 Goal I / Current

Compares representational stability in visual cortex and hippocampus using sparse manipulation of plasticity-related genes during active and passive virtual reality.

v1 ca1 experience
Project 5 - Uncertainty and Neuromodulation Goal I / Current

Tests how uncertainty-dependent behavioral states and cholinergic neuromodulation regulate representational drift across scales, including spatial transcriptomics.

uncertainty neuromodulation transcriptomics
Project 6 - Landmark Affordance Coding Goal III / Current

Quantifies how RSC and V1 encode landmark utility dimensions (stability, value, salience) in gaze-contingent closed-loop navigation tasks.

affordance landmarks navigation
Project 7 - Latent Learning Circuits Goal IV / Current

Multisensory preconditioning in virtual reality to resolve circuit mechanisms of unsupervised cue association and transfer in cortical memory systems.

latent-learning preconditioning multisensory
